Important Information

These standalone dashboards bypass the React authentication flow entirely. They are designed to work regardless of the authentication state in the React app.

Why use these? The main React app has issues with authentication redirects that are difficult to troubleshoot. These standalone pages let you access dashboard functionality without dealing with those issues.

How it works: Each page automatically sets a session token in localStorage and displays a mock dashboard interface. There is no actual connection to the database or backend services.
